# Next.js Blog

Feature-rich blog built with Next.js, MDX, and gray-matter.

## Features

- 📝 MDX support
- 🚀 Static Site Generation
- 📡 RSS feed
- 🔍 SEO optimized
- 🎨 Custom styling

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
